# Dave Gonzales Portfolio

Personal portfolio. Vue 3 + TypeScript + Tailwind, built with Vite.

The content is stored in `src/data`.

## Setup

Install dependencies:
```
npm install
```

Start the development server:
```
npm run dev
```

Build for production:
```
npm run build
```

Preview the production build locally:
```
npm run preview
```

Run type-checking:
```
npm run type-check
```
